# 🏫 ระบบเช็คชื่อและคัดกรองสุขภาพนักเรียนอัจฉริยะ (Smart Attendance System)

ระบบลงเวลาเรียนของนักเรียนผ่านกล้องเว็บแคมด้วย **ปัญญาประดิษฐ์ Face Recognition (สแกนใบหน้า) และระบบสแกน QR Code สำรอง** พร้อมด้วยระบบ **คัดกรองสุขภาพเบื้องต้น (วัดไข้/ไอ/ปกติ)** แจ้งเตือนผู้ปกครองแบบเรียลไทม์ผ่าน LINE Notify / LINE OA และระบบส่งรายงานสรุปสถิตินักเรียนของแต่ละห้องเรียนเข้ากลุ่มไลน์คุณครูและผู้ปกครองโดยอัตโนมัติประจำวัน

แอปพลิเคชันนี้ออกแบบมาให้ทำงานได้ **ฟรี 100%** ไม่มีค่าธรรมเนียมรายเดือน หรือค่าใช้จ่ายด้านเซิร์ฟเวอร์ และที่สำคัญคือ **คุณครูท่านอื่นสามารถคัดลอก (Make a Copy) นำไปตั้งค่าและใช้งานในโรงเรียนหรือห้องเรียนของตนเองได้ง่ายในเวลาไม่เกิน 5 นาที!**

---

## 🚀 ฟีเจอร์หลักของระบบ (Core Features)

1. **📸 ระบบเช็คชื่อแบบ Multi-Input (Face AI / QR / Barcode / เครื่องยิงฮาร์ดแวร์):** เช็คชื่อเข้าเรียนด้วยกล้องเว็บแคมผ่านระบบ AI จดจำใบหน้า, สแกน QR Code/Barcode ผ่านกล้อง หรือเชื่อมต่อเครื่องยิงบาร์โค้ด (USB/Bluetooth) ยิงรหัสได้ทันทีจากพื้นหลังโดยไม่ต้องกดคลิกหน้าจอ พร้อมช่องป้อนรหัสแมนนวลกรณีเครื่องยิงขัดข้อง
2. **🩺 ระบบคัดกรองสุขภาพสั้นขณะลงเวลา (Health Screening Integration):** ครูสามารถบันทึกอุณหภูมิร่างกายและสถานะคัดกรอง (ปกติ 🟢, มีอาการไอ 🟡, ตัวร้อน/มีไข้สูง 🔴) พร้อมตัวนับถอยหลัง 3 วินาทีสำหรับบันทึกอัตโนมัติเพื่อไม่ให้เกิดการต่อคิวยาวหน้าโรงเรียน ข้อมูลจะแนบไปใน LINE แจ้งเตือนทันที
3. **📊 ระบบสรุปยอดประจำวันอัตโนมัติเข้ากลุ่ม LINE (Daily Attendance Summary Bot):** ยิงข้อความรายงานยอดสถิติเข้าเรียนของวันนั้น ๆ เข้ากลุ่ม LINE ครูและผู้ปกครองโดยอัตโนมัติ ณ เวลาที่กำหนด (เช่น 08:30 น.)
4. **💳 ระบบออกบัตรนักเรียนดิจิทัล (Digital Student ID Badge):** ออกบัตรนักเรียนแบบกระจกฝ้า (Glassmorphism) มี QR Code และบาร์โค้ด (Code 128) ประจำตัวแสดงผลอยู่บนหน้าบัตร พร้อมปุ่มกดสั่งพิมพ์แยกเฉพาะตัวบัตรได้อย่างสวยงาม
5. **🚨 ระบบวิเคราะห์พฤติกรรมขาดเรียนเชิงลึก (AI Early Warning):** แจ้งเตือนเด็กกลุ่มเสี่ยงขาดเรียนบ่อย เสี่ยงหลุดออกจากระบบการศึกษา พร้อมคำแนะนำสำหรับครูประจำชั้น
6. **🔒 ระบบสิทธิ์แยกห้องเรียน (Classroom Access Control):** ล็อกสิทธิ์ครูประจำชั้นไม่ให้เห็นหรือแก้ไขห้องเรียนอื่น รวมถึงการล็อกหน้าความปลอดภัยสูงสุดสำหรับผู้ดูแลระบบ (Admin Guard)
7. **🔄 ระบบทำงานแบบออฟไลน์ (Offline Database & Auto-Sync):** หากเน็ตโรงเรียนหลุด ระบบจะยังสามารถสแกนจำหน้าเด็กในเครื่อง และจะบันทึกคิวไว้ในเบราว์เซอร์เพื่อซิงค์ส่งแจ้งเตือนย้อนหลังเข้า LINE ทันทีเมื่อต่อเน็ตติด

## 🛠️ คู่มือการติดตั้งระบบฉบับจับมือทำสำหรับคุณครู (Step-by-Step Installation Guide)

คุณครูท่านใดที่สนใจนำระบบเช็คชื่อนี้ไปใช้เป็นส่วนตัวในห้องเรียนหรือโรงเรียนของตนเอง สามารถจัดเตรียมและติดตั้งระบบทั้งหมดได้ฟรี 100% ตามขั้นตอนจับมือทำดังนี้ครับ:

### ส่วนที่ 1: เตรียมระบบฐานข้อมูล Google Sheets

1. **สร้างไฟล์แผ่นงาน Google Sheets ใหม่:** ให้คุณครูเปิด Google Sheets ของบัญชี Google ของตนเองขึ้นมา 1 ไฟล์
2. **สร้างแผ่นงานย่อย (Sheet Tabs) ทั้งหมด 4 แผ่นงาน:** ตั้งชื่อแผ่นงานย่อยด้านล่างและกรอกชื่อหัวตารางในแถวที่ 1 (Row 1) ให้ตรงตามนี้:
   * **แผ่นงานที่ 1 ตั้งชื่อว่า `students`** 
     *(หัวตารางแถวแรกในคอลัมน์ A ถึง M: `id`, `name`, `email`, `faceDescriptor`, `consentGiven`, `registeredAt`, `classroom`, `level`, `parentLineId`, `avatarUrl`, `bloodGroup`, `emergencyPhone`, `medicalAlert`)*
   * **แผ่นงานที่ 2 ตั้งชื่อว่า `attendance`** 
     *(หัวตารางแถวแรกในคอลัมน์ A ถึง J: `id`, `studentId`, `studentName`, `studentEmail`, `timestamp`, `confidence`, `classroom`, `status`, `temperature`, `healthStatus`)*
   * **แผ่นงานที่ 3 ตั้งชื่อว่า `leaves`** 
     *(หัวตารางแถวแรกในคอลัมน์ A ถึง N: `id`, `studentId`, `studentName`, `classroom`, `startDate`, `endDate`, `type`, `reason`, `status`, `submittedAt`, `contactEmail`, `contactLine`, `contactPhone`, `evidenceUrl`)*
   * **แผ่นงานที่ 4 ตั้งชื่อว่า `settings`** 
     *(หัวตารางแถวแรกในคอลัมน์ A และ B: `key`, `value`)*

3. **ติดตั้งสคริปต์ระบบ (Google Apps Script):**
   * ในหน้า Google Sheets นั้น ให้กดที่เมนู **ส่วนขยาย (Extensions) > Apps Script**
   * ระบบจะเปิดแท็บแก้ไขโค้ดขึ้นมา ให้ลบโค้ดเริ่มต้นออกให้หมด
   * คัดลอกโค้ดทั้งหมดจากไฟล์ [`google_apps_script.js`](./google_apps_script.js) ในโครงการนี้ไปวางแทนที่โค้ดเดิมทั้งหมด
   * กดปุ่ม **บันทึก (Save - ไอคอนแผ่นดิสก์)**

4. **เปิดใช้งานสคริปต์ให้เป็นเว็บแอป (Deploy Web App):**
   * คลิกที่ปุ่ม **การทำให้ใช้งานได้ (Deploy) > การทำให้ใช้งานได้ใหม่ (New Deployment)**
   * เลือกประเภทเป็น **เว็บแอป (Web App)** (คลิกไอคอนฟันเฟืองด้านซ้ายเพื่อเลือก หากยังไม่มีประเภทขึ้นมา)
   * ตั้งค่าตัวเลือกที่สำคัญดังนี้:
     * *เรียกใช้งานในฐานะ (Execute as):* เลือก **ฉัน (บัญชีอีเมล Google ของคุณครูเอง)**
     * *ผู้มีสิทธิ์เข้าถึง (Who has access):* เลือก **ทุกคน (Anyone)** *(เพื่ออนุญาตให้แอปเช็คชื่อสแกนใบหน้าบนอินเทอร์เน็ตส่งข้อมูลเข้ามาได้)*
   * คลิกปุ่ม **Deploy** จากนั้นระบบจะแสดงความปลอดภัย ให้กดปุ่ม **ให้สิทธิ์เข้าถึง (Authorize Access)** -> เลือกบัญชี Gmail -> กด **ขั้นสูง (Advanced)** -> กด **ไปที่โครงการ (ไม่ปลอดภัย)** -> กด **อนุญาต (Allow)**
   * เมื่อทำเสร็จสิ้น คัดลอกค่า **URL ของเว็บแอป (Web App URL)** (ลิงก์จะลงท้ายด้วย `/exec`) เก็บเอาไว้ใช้ในส่วนถัดไป

### ส่วนที่ 2: นำระบบเช็คชื่อขึ้นออนไลน์ผ่าน Vercel (1-Click Deploy)

คลิกปุ่มด้านล่างนี้เพื่อคัดลอกโค้ดระบบเช็คชื่อและติดตั้งออนไลน์ผ่านบัญชี Vercel ของคุณฟรีทันที:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2FJayjazzper%2FSmart-Attendance-System&env=NEXT_PUBLIC_GOOGLE_SCRIPT_URL,NEXT_PUBLIC_SCHOOL_NAME,NEXT_PUBLIC_SCHOOL_DISTRICT)

**สิ่งที่ต้องทำในหน้าต่าง Vercel Deploy:**
1. **สร้างบัญชี Vercel (ถ้ายังไม่มี):** สมัครใช้งานฟรีโดยเชื่อมต่อกับบัญชี GitHub ของคุณ
2. **สร้าง Repository ส่วนตัว:** ป้อนชื่อโปรเจกต์ (เช่น `smart-attendance-system`) Vercel จะสร้างสำเนาซอร์สโค้ดนี้ส่งเข้าไปยังบัญชี GitHub ของคุณโดยอัตโนมัติ
3. **ป้อนค่าตัวแปรสภาพแวดล้อม (Environment Variables):**
   * `NEXT_PUBLIC_GOOGLE_SCRIPT_URL`: วางลิงก์ **URL ของเว็บแอป** ที่คัดลอกมาจากขั้นตอน Google Sheets ด้านบน (ลิงก์ลงท้ายด้วย `/exec`)
   * `NEXT_PUBLIC_SCHOOL_NAME`: ป้อนชื่อโรงเรียนของคุณ (เพื่อแสดงผลบนหัวเว็บ)
   * `NEXT_PUBLIC_SCHOOL_DISTRICT`: ป้อนสังกัด/สำนักงานการศึกษาของโรงเรียนคุณ
4. คลิกปุ่ม **Deploy**
5. นั่งรอระบบประมวลผลประมาณ 1 นาที ระบบจะแจ้งเตือนสำเร็จและให้ลิงก์ออนไลน์อย่างเป็นทางการสำหรับเข้าใช้งานผ่านมือถือ แท็บเล็ต หรือเครื่องคอมพิวเตอร์ของคุณครูได้ทันทีตลอด 24 ชั่วโมง!

### ส่วนที่ 3: ขั้นตอนการตั้งค่ากลุ่มไลน์แจ้งเตือนผู้ปกครอง (LINE Notify)

เมื่อติดตั้งเว็บเสร็จแล้ว แอดมินต้องทำการผูกรหัสกลุ่มแจ้งเตือนไลน์ให้กับห้องเรียนแต่ละห้อง:
1. **เข้าสู่ระบบหลังบ้าน:** ไปที่หน้าเว็บโรงเรียนของคุณครู -> คลิกที่ **เครื่องมือระบบ (ฟันเฟือง)** -> เลือก **ตั้งค่าระบบ** (ล็อกอินด้วย Username: `admin` และ Password เริ่มต้นคือ `1234`)
2. **ออกโทเคนจาก LINE Notify:**
   * เปิดแท็บใหม่ เข้าไปที่เว็บ [LINE Notify My Page](https://notify-bot.line.me/my/) แล้วล็อกอินด้วยบัญชี LINE
   * เลื่อนลงมาล่างสุด กดปุ่ม **ออกโทเคน (Generate Token)**
   * ตั้งชื่อบริการแสดงตัวบอทแจ้งเตือน (เช่น *เช็คชื่อ ป.4/1*) จากนั้นค้นหากลุ่ม LINE ที่ต้องการส่งข้อความแจ้งเตือน แล้วคลิกเลือกกลุ่มนั้น
   * กดปุ่ม **ออกโทเคน (Generate Token)** และ**คัดลอกรหัส (Token)** เก็บเอาไว้
3. **เชิญบอทเข้ากลุ่มแชท:** เปิดแอป LINE ในมือถือ แล้วเชิญบัญชีบอทที่ชื่อว่า **`LINE Notify`** เข้ามาในกลุ่มแชทไลน์ดังกล่าว
4. **บันทึกรหัสในระบบ:** นำรหัส Token ที่ได้มาวางลงในช่องห้องเรียนในหน้าตั้งค่าระบบของเว็บแอป และกดบันทึก คุณครูสามารถกดปุ่ม **ทดสอบส่งข้อความ** เพื่อทดสอบเชื่อมสัญญาณได้ทันที

## 🔧 การพัฒนาและรันบนเครื่องคอมพิวเตอร์ท้องถิ่น (Local Development)

หากต้องการรันเพื่อแก้ไขดีไซน์หรือพัฒนาโค้ดเพิ่มเติมในเครื่องคอมพิวเตอร์ของตัวเอง:

1. ติดตั้งไลบรารีทั้งหมดในไดเรกทอรีโครงการ:
   ```bash
   npm install
   ```
2. ป้อนค่าตัวแปรสภาพแวดล้อมในไฟล์ `.env.local`:
   ```env
   NEXT_PUBLIC_GOOGLE_SCRIPT_URL=ลิงก์เว็บแอป_ของสคริปต์
   ```
3. สั่งรันเวอร์ชันทดสอบบนเครื่อง:
   ```bash
   npm run dev
   ```
4. เปิดเบราว์เซอร์เข้าที่: [http://localhost:3000](http://localhost:3000)
